## рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ 29.07.2022

1) рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЎрЃџрЃљрЃАрЃў FacialEmotion, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃљрЃЦрЃЋрЃА рЃЮрЃарЃў рЃЋрЃћрЃџрЃў: eyebrows рЃЊрЃљ mouth. eyebrows-рЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃЮ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃљ -> "upward", "downward", "raised" рЃЊрЃљ "neutral", рЃ«рЃЮрЃџрЃЮ mouth-рЃўрЃА -> "smile", "frown", "laugh" рЃЊрЃљ "neutral". рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃљрЃЏ рЃЮрЃа рЃЋрЃћрЃџрЃА, рЃњрЃљрЃарЃЊрЃљ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃўрЃАрЃљ, рЃАрЃ«рЃЋрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃЋрЃћрЃа рЃЏрЃўрЃћрЃюрЃўрЃГрЃћрЃЉрЃљ .

2) рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃарЃњрЃБрЃЏрЃћрЃюрЃбрЃљрЃЊ рЃўрЃдрЃћрЃЉрЃА FacialEmotion рЃЎрЃџрЃљрЃАрЃўрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃА, рЃЊрЃљ рЃЏрЃўрЃАрЃњрЃљрЃю рЃЦрЃЏрЃюрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃБрЃа JSON рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃА.

3) рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ рЃцрЃБрЃюрЃфрЃЦрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃарЃњрЃБрЃЏрЃћрЃюрЃбрЃљрЃЊ рЃЏрЃўрЃўрЃдрЃћрЃЉрЃА рЃЏрЃћрЃЮрЃарЃћ рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃўрЃЊрЃљрЃю рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџ String-рЃА, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф JSON-рЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃў рЃБрЃюрЃЊрЃљ рЃ░рЃЦрЃЮрЃЊрЃћрЃА. рЃљрЃЏ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃерЃў рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЌ рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃБрЃџрЃў String-рЃўрЃА рЃЊрЃћрЃАрЃћрЃарЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃљрЃ«рЃљрЃџрЃў рЃЎрЃџрЃљрЃАрЃўрЃА, FacialEmotionRecognized рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃерЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф, рЃњрЃљрЃарЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЮрЃарЃў рЃЋрЃћрЃџрЃўрЃАрЃљ (eyebrows, mouth), рЃљрЃЦрЃЋрЃА рЃЏрЃћрЃАрЃљрЃЏрЃћрЃф - emotion. рЃљрЃЏ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃЏ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЮрЃарЃў рЃЋрЃћрЃџрЃўрЃА, рЃгрЃљрЃарЃЉрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃърЃўрЃарЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃЌ рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃљрЃЊрЃњрЃўрЃюрЃЮрЃА рЃАрЃљрЃ«рЃўрЃА рЃћрЃЏрЃЮрЃфрЃўрЃљ, рЃЊрЃљ рЃАрЃўрЃбрЃДрЃЋрЃўрЃћрЃарЃљрЃЊ рЃЕрЃљрЃгрЃћрЃарЃЮрЃА emotion рЃЋрЃћрЃџрЃерЃў, рЃАрЃ«рЃЋрЃљ рЃЮрЃа рЃЋрЃћрЃџрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ. рЃЌрЃБ рЃърЃўрЃарЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљ рЃљрЃарЃўрЃА рЃюрЃћрЃўрЃбрЃарЃљрЃџрЃБрЃарЃў, рЃЊрЃљ рЃгрЃљрЃарЃЉрЃћрЃЉрЃўрЃАрЃљрЃф (­ЪўЉ), рЃЕрЃљрЃЋрЃЌрЃЋрЃљрЃџрЃЮрЃЌ, рЃарЃЮрЃЏ рЃћрЃЏрЃЮрЃфрЃўрЃљ рЃљрЃарЃўрЃА рЃюрЃћрЃўрЃбрЃарЃљрЃџрЃБрЃарЃў, рЃЊрЃљ рЃЌрЃБ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃгрЃљрЃарЃЉрЃћрЃЉрЃў рЃљрЃарЃўрЃА downward, рЃ«рЃЮрЃџрЃЮ рЃърЃўрЃарЃў smile, рЃЋрЃЌрЃЋрЃљрЃЌ, рЃарЃЮрЃЏ рЃћрЃА рЃљрЃарЃўрЃА рЃЉрЃЮрЃарЃЮрЃбрЃў рЃдрЃўрЃЏрЃўрЃџрЃў >:D, рЃЊрЃљ рЃљ.рЃе.

4) рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЌ FacialEmotionRecognized рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃАрЃћрЃарЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃф.

* рЃАрЃћрЃарЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ/рЃЊрЃћрЃАрЃћрЃарЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ Gson рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ.
